## 📌 Course Overview

This repository contains all lab submissions for **Principles of Communication Systems**. The labs focus on practical implementation of communication concepts using:

- **NI LabVIEW** graphical programming.
- **USRP** (Universal Software Radio Peripheral) / Software Defined Radio.
- Analog and Digital Modulation techniques.

---

## 🛠️ Labs Covered

| Lab # | Title |
|-------|-------|
| Lab 01 | Array Control, MathScript Functions & Temperature Conversion using LabVIEW |
| Lab 02 | Array Manipulation, While Loop Programming & Introduction to SDR |
| Lab 03 | Amplitude Modulation |
| Lab 04 | Continuous Waveform Transmission and Receiving using USRP |
| Lab 05 | Amplitude Modulation (Full Carrier) Implementation using USRP |
| Lab 06 | Amplitude Modulation Suppressed Carrier (SC) using USRP |
| Lab 07 | Frequency Modulation |
| Lab 08 | Sampling Theorem |
| Lab 09 | Line Coding Techniques |
| Lab 10 | Amplitude Shift Keying (ASK) |
